# ytmpd (archived)

This project has been superseded by [xmpd](https://github.com/tuncenator/xmpd),
a multi-provider rewrite supporting YouTube Music, Tidal HiFi, and more.

All development continues there. This repository is archived and will not
receive updates.

## Migrating

See [xmpd docs/MIGRATION.md](https://github.com/tuncenator/xmpd/blob/main/docs/MIGRATION.md)
for the `ytmpd` to `xmpd` migration guide covering the config shape change,
renamed binaries (`ytmpctl` to `xmpctl`, `ytmpd-status` to `xmpd-status`),
and the new multi-source provider architecture.
